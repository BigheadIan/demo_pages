# 旅遊客服工作台 DEMO

這是一個靜態的旅遊業客服工作台 DEMO 網站，模擬即時通訊軟體的介面，展示客服人員的日常工作流程。

## 功能特色

### 左側客戶列表區
- 顯示所有聯絡的客戶（7位模擬客戶）
- 顯示客戶頭像、暱稱、聯絡來源（LINE、FB、官網）
- 顯示所屬公司（聯發科、台積電、言智）
- 顯示最後一則訊息與時間
- 支援客戶搜尋功能
- 點擊客戶即可切換對話

### 中間對話區
- **頂部資訊欄**：顯示對話人的頭像、姓名、公司、來源、線上狀態
- **對話訊息區**：
  - 客戶訊息顯示在左側（白色背景）
  - 客服訊息顯示在右側（藍色背景）
  - 顯示每則訊息的時間戳記
  - 自動捲動到最新訊息
- **輸入框工具列**（圖示按鈕）：
  - 😊 **表情符號**：點擊顯示表情選擇器，可插入表情到訊息中
  - 📎 **傳送文件**：模擬功能
  - 📷 **傳送照片**：模擬功能
  - 📸 **截圖工具**：模擬功能
  - 💬 **罐頭訊息**：點擊顯示常用回覆模板，可直接插入對話
- **發送按鈕**：發送訊息（或按 Enter 鍵）

### 右側快速查詢區
- **服務類型選擇**：機票、住宿、鐵路、預約接送、租車、全部
- **客戶需求輸入框**：輸入查詢條件
- **查詢按鈕**：執行查詢
- **結果顯示**：
  - 以卡片形式展示查詢結果
  - 顯示服務名稱、價格、詳細資訊
  - 每個結果都有「插入對話」按鈕，可將資訊快速插入到對話輸入框

## 技術架構

- **純 HTML/CSS/JavaScript**：無需任何建置工具或伺服器
- **響應式設計**：支援桌面和平板裝置
- **現代簡約風格**：靈感來自 Slack 和 Microsoft Teams
- **模擬數據**：所有數據都使用 JavaScript 定義的靜態數據

## 如何使用

1. 將所有檔案下載到同一個資料夾
2. 直接開啟 `index.html`（使用任何現代瀏覽器）
3. 無需安裝任何工具或啟動伺服器

## 互動功能說明

### 選擇客戶
- 點擊左側任一客戶即可查看該客戶的對話記錄
- 可以在頂部搜尋框輸入關鍵字篩選客戶

### 發送訊息
- 在底部輸入框輸入文字
- 點擊「發送」按鈕或按 Enter 鍵發送
- 系統會模擬客戶自動回覆（約1秒後）

### 使用工具
- **表情符號**：點擊表情圖示，選擇表情後會插入到輸入框
- **罐頭訊息**：點擊罐頭訊息圖示，選擇常用回覆模板
- 其他工具為模擬功能（彈出提示）

### 快速查詢
1. 在下拉選單選擇服務類型
2. 在需求輸入框輸入查詢條件
3. 點擊「查詢」按鈕
4. 查看右側顯示的查詢結果
5. 點擊「插入對話」可將結果插入對話框

## 模擬數據說明

### 客戶數據
- 7位模擬客戶
- 包含聯發科、台積電、言智等公司的客戶
